# Feature Engineering on Housing Dataset

## Objective
The objective of this project is to perform feature engineering techniques on a housing dataset to improve data quality and prepare the data for machine learning models.

## Tools and Libraries Used
- Python
- Pandas
- NumPy

## Dataset
Housing Dataset (Housing.csv)

## Steps Performed
1. Loaded the dataset using Pandas.
2. Explored the dataset and checked column information.
3. Created new features:
   - Area_per_Bedroom
   - Total_Rooms
   - Price_per_Area
4. Handled missing values.
5. Applied One-Hot Encoding to categorical features.
6. Applied Log Transformation on the price column.
7. Saved the enhanced dataset.
8. Viewed the final transformed dataset.

## Feature Engineering Techniques Used
- Feature Creation
- Missing Value Handling
- One-Hot Encoding
- Log Transformation

## Output
An enhanced housing dataset with additional engineered features suitable for machine learning applications.
